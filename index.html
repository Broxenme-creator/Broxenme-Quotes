Home Page

The landing page features a Daily Inspirational Quote that changes with each click of the refresh button. Quotes are fetched from quotable.io with local fallbacks. Below you'll find Trending Quotes - the most-liked quotes across the platform.

Reels Page

Full-screen, TikTok-style quote browsing:

  • Scroll vertically between reels (snap-to behavior)
  • Like, comment, share, and save using sidebar buttons
  • Tap user profile picture to view their profile
  • Transparent search bar at top filters reels in real-time
  • Comments panel slides up - reply, like/dislike comments
  • Creator badge shows when post author interacts in comments
  • Song display with spinning disc icon when a song is attached

Community Page

Feed Tab

  • Create Posts with the Advanced Creator (text, images, custom backgrounds)
  • Like and comment on feed posts
  • Click comments to open comment popup - reply, like/dislike comments
  • Creator badge shows when post author comments

Groups Tab

  • Search and join groups - find communities by topic
  • Create new groups with name, description, and goal
  • WhatsApp-style chat - real-time messaging with sent/received styling
  • Group details (info button) - members list with follow buttons
  • Unread badges show messages from others (excludes your own)
  • Leave/Join groups freely

Discover Page

Find other users on the platform:

  • Search by name, bio, location, OR tags
  • Filter by category (Motivation, Business, Love, etc.)
  • Results show user profiles with follow buttons and tag chips
  • Click a user to view their full profile

Profile Page

  • Edit profile - name, bio, location, website, profile picture
  • Streak counter tracks daily visits (resets if you miss a day)
  • Stats in one line - Posts, Followers, Following, Total Likes
  • My Tags dropdown - select tags for better discoverability
  • Post button opens Advanced Creator (InShot/CapCut style)
  • Tabs with badges: Posts, Saved, Feeds, Groups
  • Groups tab shows all your groups with unread message counts

Advanced Post Creator

Professional editing interface:

  • Draggable text - position quote anywhere on canvas
  • 14+ fonts, 14+ text colors, 14+ background gradients
  • Upload background image
  • Crop tools - 1:1, 4:5, 9:16, 16:9 ratios
  • Text size slider (12px-72px)
  • 8 border styles for text decoration
  • Post type selector - Reel or Feed
  • Tag selector - add hashtags
